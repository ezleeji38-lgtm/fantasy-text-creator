## 공통 스타일 가이드 (모든 prompt 첫 부분에 그대로 사용)

```
Disney Pixar 3D animation movie still, in the style of Frozen, Tangled, or Encanto.

expressive characters with soft warm lighting. NOT anime, NOT manga, NOT
photorealistic. Smooth 3D-rendered surfaces, painterly textures, atmospheric.
```

## §2. 사용 워크플로우 (Frozen 스타일 적용 후)

### ChatGPT.com (GPT-4o 추천)
1. 새 채팅 → GPT-4o 선택
2. **첫 메시지에 "공통 스타일 가이드" + "캐릭터 가이드"를 한 번 알려주기**
   (그러면 14번 반복 안 해도 됨 — ChatGPT가 기억)
3. 그 후 각 장면 prompt만 차례로 입력
4. 생성된 이미지 다운로드 → `images/` 폴더 저장 (파일명 §1 표 기준)

### Bing Image Creator
스타일 가이드 + 장면을 한 prompt 안에 합쳐서 입력 (Bing은 기억 안 함).

### 도구 무관 — 결과물이 너무 medieval/anime 스타일로 나올 때
프롬프트에 추가:
- "Disney Pixar 3D animation, NOT anime, NOT medieval realism, NOT photorealistic"
- "Like Frozen / Tangled / Encanto movie still"
- "Modern teenage girl in contemporary clothes (blue hoodie, jeans, sneakers)"
