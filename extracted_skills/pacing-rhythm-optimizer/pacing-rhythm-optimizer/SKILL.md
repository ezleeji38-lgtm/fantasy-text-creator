# Pacing & Rhythm Optimizer (페이싱 및 리듬 최적화기)

판타지 소설의 페이싱을 분석하고 최적화하는 도구. 독자가 책을 놓을 수 없다는 경험은 페이싱의 산물이다.

핵심 원칙: 페이싱은 속도가 아니라 리듬이다. 빠른 것이 좋은 것이 아니고, 느린 것이 나쁜 것이 아니다. 긴장(tension)과 이완(release)의 교대, 그리고 전체적인 에스컬레이션이 핵심이다. 음악에서 빠른 악장과 느린 악장이 교대하듯, 소설에서도 액션 시퀀스와 성찰/관계 장면이 교대해야 한다.

---

## 전체 흐름

```
Step 1: 긴장-이완 곡선 분석 (Tension Curve Analysis)
Step 2: 정보 밀도 분석 (Information Density)
Step 3: 감정 곡선 분석 (Emotional Arc Analysis)
Step 4: 페이싱 문제 진단과 처방 (Problem Diagnosis)
Step 5: 장르별 페이싱 가이드 (Genre-Specific Pacing)
Step 6: 페이싱 리포트 출력 (Pacing Report)
```

---

## Step 1: 긴장-이완 곡선 분석 (Tension Curve Analysis)

이야기 전체의 긴장 수준을 챕터별로 매핑한다. 각 챕터의 긴장도를 1-10으로 평가한다.

검증 기준:
- 긴장의 전반적 상승 곡선이 있는가 (에스컬레이션)
- 긴장 후 이완이 있는가 (독자에게 숨 쉴 틈)
- 이완 구간이 3챕터 이상 연속되지 않는가 (처짐 위험)
- 클라이맥스가 최고 긴장점인가
- 클라이맥스 이후 급격한 이완이 있는가 (해결 구간)
- 1막의 긴장이 충분히 빠르게 확립되는가 (독자 유지)

긴장 곡선의 이상적 형태:
```
높음 |           /\        /\    /\  /\
     |          /  \      /  \  /  \/  \
     |    /\   /    \    /    \/        \
     |   /  \ /      \  /               \
낮음 |  /    V        \/                 \___
     +---1막---+---2막 전반---+---2막 후반---+--3막--
```

---

## Step 2: 정보 밀도 분석 (Information Density)

챕터별 새로운 정보(세계관, 캐릭터, 플롯)의 밀도를 분석한다.

과밀: 한 챕터에 새로운 고유명사 5개 이상, 새로운 규칙 2개 이상, 새로운 캐릭터 3명 이상이 동시에 등장하면 독자의 소화 능력을 초과한다.

과소: 3챕터 이상 새로운 정보/발견/변화 없이 진행되면 독자가 이탈 위험.

적정: 각 챕터에 1-2개의 새로운 정보/발견/변화. 독자가 소화하면서도 진행이 느껴지는 밀도.

정보 밀도의 에스컬레이션: 이야기 초반에는 정보 밀도를 낮게 유지하고(세계에 적응 시간), 중반부터 높이며, 클라이맥스 직전에 최대치에 도달한 후, 해결 구간에서 급격히 낮춘다.

---

## Step 3: 감정 곡선 분석 (Emotional Arc Analysis)

Matthew Jockers와 Andrew Reagan의 연구(2016, EPJ Data Science)에서 도출된 6가지 감정 아크 패턴:

1. 상승형 (Rags to Riches): 지속적 상승
2. 하강형 (Tragedy): 지속적 하강
3. 하강-상승형 (Man in a Hole): 구덩이에 빠졌다 올라옴
4. 상승-하강형 (Icarus): 올라갔다 추락
5. 상승-하강-상승형 (Cinderella): 올라갔다 떨어졌다 다시 올라옴
6. 하강-상승-하강형 (Oedipus): 떨어졌다 올라갔다 다시 떨어짐

이 연구에서 다운로드 수 기준으로 가장 인기 있는 패턴은 3번(하강-상승)과 5번(상승-하강-상승)이었다. 대부분의 베스트셀러 판타지는 5번(신데렐라) 패턴을 따른다: 주인공이 능력을 얻어 상승하고, 중반의 패배로 추락하며, 최종 성장을 통해 다시 상승한다.

## Step 4: 페이싱 문제 진단과 처방 (Problem Diagnosis)

| 증상 | 원인 | 처방 |
|------|------|------|
| 시작이 느림 | 과도한 세계관 설명 / 갈등 부재 | In medias res(사건 중간에서 시작) 또는 갈등 선행. 세계관 설명은 갈등 속에 녹임 |
| 중반부 처짐 (Sagging Middle) | 갈등 에스컬레이션 부재 / 방향 상실 | 중간점에 큰 반전/발견 배치. 서브플롯 활성화. 새로운 위협 도입 |
| 3/4 지점 탈진 | 가장 어두운 순간 이후 회복이 느림 | 3막 진입을 빠르게 가속. 불필요한 성찰 장면 축소 |
| 클라이맥스가 약함 | 스테이크 부족 / 감정 축적 부족 | 개인적+우주적 스테이크 동시 배치. 관계적 판돈 추가 |
| 전체적으로 단조 | 긴장-이완 리듬 부재 | 액션 시퀀스 후 조용한 캐릭터 장면 삽입. 유형 다른 장면 교대 |
| 전체적으로 너무 빠름 | 감정적 순간의 부재 / Sequel 부족 | 중요 순간에 시간 늘리기. 반응-딜레마-결정(Sequel)의 비중 증가 |
| 전체적으로 너무 느림 | 갈등 부족 / 불필요한 장면 | 목적 불분명한 장면 삭제. 각 장면에 갈등 요소 추가 |
| 에필로그 지루함 | 클라이맥스 후 너무 많은 해결 | 에필로그를 1-2장면으로 압축. 여운은 짧을수록 강력 |

### 중반부 처짐(Sagging Middle) 특화 처방

에픽 판타지에서 가장 흔한 문제이다. 처방 순서:

1. 중간점에 판도를 뒤집는 사건을 배치 (반전, 배신, 폭로)
2. 서브플롯을 활성화하여 다양한 갈등을 교대로 제시
3. 새로운 캐릭터나 세력을 도입하여 판도를 복잡화
4. 시간 압박을 도입하여 긴박감 생성
5. 개인적 스테이크를 상승시킴 (사랑하는 사람의 위험 등)

---

## Step 5: 장르별 페이싱 가이드 (Genre-Specific Pacing)

에픽 판타지: 느린 축적과 폭발적 해소의 리듬. 초반 2-3챕터에서 세계와 캐릭터를 확립하되, 최소 1개의 갈등 씨앗을 심는다. 중반부가 가장 긴 구간이며, 서브플롯의 교대가 처짐을 방지한다. 클라이맥스는 물리적 전투와 감정적 해결이 동시에.

다크 판타지: 일관되게 높은 긴장. 이완 구간이 짧고, 안전한 순간이 없다는 느낌. 유머나 따뜻한 순간이 있을 때 오히려 독자가 불안해하는 효과.

한국 웹소설: 빠른 순환 리듬. 매 챕터가 미니 아크(도전-시련-성장/보상)를 가짐. 긴 축적보다 빈번한 보상(레벨업, 스킬 획득, 승리)으로 독자를 유지. 대규모 아크(10-20챕터)와 소규모 아크(2-5챕터)의 중첩.

어반 판타지: 미스터리/스릴러의 페이싱 차용. 정보의 점진적 공개가 추진력. 액션 장면은 짧고 폭발적. 캐릭터의 사생활과 초자연적 사건의 교대.

## Step 6: 페이싱 리포트 출력 (Pacing Report)

```
페이싱 분석 리포트
━━━━━━━━━━━━

원고: [제목]

[긴장-이완 곡선]
챕터별 긴장도 수치 (1-10)와 시각적 표현

[정보 밀도 분석]
챕터별 새 정보 유형과 수량

[감정 곡선 패턴]
전체 감정 아크 유형 판별

[문제 지점]
처짐/과속/단조 구간 식별

[처방]
각 문제에 대한 구체적 처방

[장르 기준 대비]
현재 페이싱이 장르 관습에 부합하는지
```
