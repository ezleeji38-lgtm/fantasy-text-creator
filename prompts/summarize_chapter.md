방금 집필된 챕터를 다음 챕터 집필 시 참고할 **연속성 요약**으로 압축하시오.

# 요약 원칙

- 분량: **500~800자**
- 시점: 사건이 일어난 순서대로
- 반드시 포함: 장소, 시간, 등장한 인물, 핵심 사건, 인물 간 관계 변화, 새로 등장한 설정·아이템, **미해결 상태**(열린 질문·긴장)
- 생략: 배경 묘사, 감정 묘사, 대사 원문

# 출력 형식 (마크다운)

```
## 장소·시간
- ...

## 등장인물
- 인물A: 상태/행동
- ...

## 핵심 사건
1. ...
2. ...

## 관계 변화
- ...

## 새 설정·아이템
- ...

## 미해결 상태 / 다음 챕터로 이어질 것
- ...
```

코드블록 펜스 없이 위 구조만 출력하시오.
