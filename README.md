설치 방법은 다음과 같습니다. 
각 .skill 파일을 다운로드한 후, Claude 프로젝트의 설정(Settings)에서 "Skills" 또는 "Custom Skills" 메뉴로 들어가 .skill 파일을 업로드하면 됩니다. 
파일을 하나씩 업로드해도 되고, 모두 다운로드한 후 순서대로 설치해도 됩니다.
설치 순서는 Phase 순서를 따르는 것을 권장합니다. 
Phase 1(fantasy-worldbuilder, species-culture-designer, magic-system-architect)을 먼저, 
이어서 Phase 2(character-arc-engineer, relationship-web-mapper), 
Phase 3(plot-architecture-engine, conflict-tension-designer, foreshadow-twist-weaver), 
Phase 4(prose-style-engine, dialogue-voice-engine, action-battle-choreographer, scene-chapter-composer), 
Phase 5(consistency-auditor, pacing-rhythm-optimizer, beta-reader-simulator-v2, revision-rewrite-engine), 
Phase 6(series-expansion-planner, fantasy-novel-assembler) 순서입니다.
